Oficiální API

Tato webová aplikace získává data pomocí knihovny bakalari (Dart). Tato knihovna je veřejně k dispozici, zcela zdarma. Zdrojový kód je k dispozici na GitHubu.

Existují i další knihovny, například knihovna pybakalib pro python, ze které má Dart knihovna vychází.

API školního systému Bakaláři je shrnuto v tomto repozitáři.

Neoficiální API

Data můžete získávat i z této webové aplikace.

Login

POST /student/json
x-www-webform-urlencoded
  • bakawebUrl (String)
  • login (String)
  • password (String)

Odpověď: GUID, které je nutné pro přístup k informacím o studentovi.

GetContent

GET /student/json
x-www-webform-urlencoded
  • studentID (String)

Vstup: studentID je GUID, které jde získat z předchozího endpointu.

Výstup: Kompletní údaje o studentovi (JSON). Tyto informace chvíli trvá získat, takže je dobrý nápad chvíli po přihlášení počkat. Je možné tuto žádost poslat několikrát, pokaždé se odešlou zpátky všechny informace, které jsou k dispozici. Jakmile server vrátí 201 (Created), už vše doběhlo a nemá cenu tento endpoint znovu volat.
